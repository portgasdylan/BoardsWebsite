# Portfolio Website - UI Wireframes & Design Documentation

## Overview
This document provides wireframes and design specifications for the Dylan Maxwell Portfolio Website, demonstrating the user interface design process from concept to implementation.

## 1. Design Philosophy

### Visual Theme
- **Manga/Anime Inspired**: Bold geometric shapes, dynamic angles, and high contrast
- **Professional Portfolio**: Clean, readable, and business-appropriate
- **Modern Web Design**: Responsive, accessible, and performance-optimized

### Color Palette
- **Primary**: Black (#000000) - Professional, bold
- **Accent**: Red (#DC2626) - Energy, passion, attention-grabbing
- **Background**: White (#FFFFFF) and Light Gray (#F3F4F6)
- **Text**: Dark Gray (#374151) for readability

### Typography
- **Headers**: Bold, uppercase for impact
- **Body**: Clean, readable sans-serif
- **Code**: Monospace font for technical content

## 6. Animation Specifications

### Scroll Animations
- **Fade In Up**: Elements slide up 50px while fading in
- **Slide In Left**: Elements slide in from left (-50px)
- **Slide In Right**: Elements slide in from right (50px)
- **Duration**: 0.6 seconds with easing
- **Trigger**: When element is 10% visible in viewport

### Micro-interactions
- **Button Hover**: Scale 1.05, duration 0.3s
- **Card Hover**: Scale 1.02, shadow increase, duration 0.3s
- **Link Hover**: Color change to red, duration 0.3s

## 7. Accessibility Considerations

### Color Contrast
- **Text on White**: Black (#000000) - AAA compliant
- **Text on Black**: White (#FFFFFF) - AAA compliant
- **Red Accent**: Used sparingly, never for essential text

### Navigation
- **Keyboard Navigation**: All interactive elements accessible via Tab
- **Screen Reader**: Semantic HTML with proper ARIA labels
- **Focus Indicators**: Clear visual focus states

### Responsive Text
- **Mobile**: 16px minimum font size
- **Tablet**: 18px base font size
- **Desktop**: 20px base font size
